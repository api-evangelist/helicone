# Helicone (helicone)
Helicone is an open-source LLM observability platform and AI gateway that helps developers monitor, evaluate, and experiment with their AI applications. The worlds fastest-growing AI companies rely on Helicone to route, debug, and analyze their applications. With a single line of code, Helicone provides unified request logging, cost tracking, latency analysis, prompt management, experiments, evaluations, session tracing, caching, rate limiting, and webhooks across 100+ AI models from providers including OpenAI, Anthropic, Google, Groq, and more. The platform offers an OpenAI-compatible AI Gateway with automatic provider fallbacks, zero markup pricing, and built-in observability. Helicone is SOC 2 and GDPR compliant, supports self-hosting via Docker and Kubernetes, and offers a generous free tier of 10,000 requests per month.
